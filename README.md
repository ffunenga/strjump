Checkout [http://ffunenga.github.io/strjump](http://ffunenga.github.io/strjump) for more information.

**Install** strjump from PyPI with:
```
$ pip install strjump
```

**Test** strjump with:

```
$ python -m unittest discover
```

**Deploy** to PyPI with:

```
$ python setup.py sdist bdist_wheel
$ twine upload dist/*
```
